## Details

### 0.1.x — First public release

Core form components, native form styling, Symfony Form integration, registry alignment

### 0.2.x — Form foundations

Additional field variants, theming improvements, metadata integration, component consistency

### 0.3.x — Additional form controls

Choice controls, grouped inputs, file upload, search fields, range controls, color picker

### 0.4.x — Composite widgets

Input groups, segmented controls, password widgets, autocomplete patterns, date and time compounds

### 0.5.x — Validation and accessibility

Validation states, helper messages, error presentation, ARIA refinements, keyboard navigation

### 0.6.x — Layouts and collections

Collection fields, repeatable groups, inline layouts, responsive form layouts, form sections

### 0.7.x — Advanced form patterns

Multi-step forms, conditional sections, progressive disclosure, confirmation patterns, interaction refinements

### 0.8.x — Documentation

Complete examples, Symfony Form recipes, integration guides, migration guides, design recommendations

### 0.9.x — API stabilization

Component review, naming consistency, deprecation cleanup, documentation polish, compatibility validation

### 1.0.x — Stable API

Semver-stable component API, stable theme contracts, stable extension points, complete handbook
